About Me

I'm Underscore - I've been scripting on Roblox for about 3 years now. I've been doing game design for 2 years while I was getting into the coding side, which honestly helped a lot. Knowing how a game is supposed to feel makes it easier to script it right.

I mostly work on the backend stuff - systems that run the game, handle security, keep things from breaking. From past experience, building things in pieces that actually slot together instead of one giant mess of a script works better.

I'm not gonna lie, my stuff doesn't always look the prettiest but it works, and it holds up. I've shipped games solo, led small teams, and done a few commissions. If you need something built properly, I could be the right person.
